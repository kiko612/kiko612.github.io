VS2019安装Qt插件教程，发现下载不了问题解决

技术标签： 工具类 window visual studio

1.打开VS，最上方工具栏中点击扩展窗口，选择管理扩展

2.在右边搜索中搜索qt出现以下界面，这时可能出现问题：再点击下载发现迟迟下载不了，或者是下载到一定地步后无法下载，再或者是下载完成后安装无反应
解决办法：
点击有点的详细信息或者进入官网直接下载qt插件进行安装；官网太慢的话，可以去百度网盘,提取码：ix7i，下载。


3.点击下载并进行安装
在安装过程中一定要把Microsoft Visual Studio 2019关闭程序。否则会卡在以下界面：




安装完成后重新启动VS发现新建项目

这样就OK啦
